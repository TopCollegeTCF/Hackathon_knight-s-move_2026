О хакатоне

Создайте свою игру в жанре Tower Defence

Задача

За 13 дней создать работающую браузерную игру Tower Defence с уникальной механикой, используя только HTML, CSS и JavaScript.

Командная работа

Работа в командах 2-4 человека с использованием Git и GitHub. Каждый участник вносит свой вклад в общий проект.

Награды

Все участники получат 50 топ коинов, а победители - 100 топ коинов и специальные оценки по предметам.
